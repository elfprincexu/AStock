## 1. 实时行情接口

**URL:** `https://push2.eastmoney.com/api/qt/stock/get`

### 1.1 请求参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `secid` | `1.600519` (SH) 或 `0.000001` (SZ) | 市场代码.股票代码。`1` = 上交所 (SH)，`0` = 深交所 (SZ) |
| `fields` | `f43,f44,f45,...` | 需要返回的字段列表，逗号分隔 |
| `ut` | `fa5fd1943c7b386f172d6893dbbd1177` | 请求令牌 (公开通用值) |

### 1.2 返回字段详解

响应 JSON 结构: `{ "data": { "f43": ..., "f44": ..., ... } }`

#### 价格类字段

> **关键规则:** 价格类字段返回的是**原始整数**，需要根据 `f59` (小数精度) 进行除法换算。
>
> `实际价格 = 原始值 / (10 ^ f59)`
>
> 例: `f59=2` 时，`f43=152300` 表示价格 `1523.00`；`f59=3` 时，`f43=15230` 表示 `15.230`。

| 字段 | 中文名 | 原始值类型 | 转换公式 | 说明 |
|------|--------|-----------|----------|------|
| `f43` | **最新价** (current price) | 整数 | `÷ 10^f59` | 当前最新成交价。非交易时段为上一交易日收盘价 |
| `f44` | **最高价** (high) | 整数 | `÷ 10^f59` | 当日最高成交价 |
| `f45` | **最低价** (low) | 整数 | `÷ 10^f59` | 当日最低成交价 |
| `f46` | **开盘价** (open) | 整数 | `÷ 10^f59` | 当日开盘价 |
| `f60` | **昨收价** (previous close) | 整数 | `÷ 10^f59` | 上一交易日收盘价，用于计算涨跌幅 |
| `f59` | **小数精度** (decimal precision) | 整数 | — | 价格字段的小数位数。通常 A 股为 `2` |

#### 成交量价字段

| 字段 | 中文名 | 原始值类型 | 单位 | 转换规则 | 说明 |
|------|--------|-----------|------|----------|------|
| `f47` | **成交量** (volume) | 整数 | **手** | `× 100 = 股` | 1 手 = 100 股。例: `f47=50000` 表示 5,000,000 股 |
| `f48` | **成交额** (amount/turnover) | 浮点数 | **元** | 直接使用 | 当日累计成交金额 (人民币元) |

#### 涨跌与换手字段

| 字段 | 中文名 | 原始值类型 | 转换公式 | 说明 |
|------|--------|-----------|----------|------|
| `f170` | **涨跌幅** (change percent) | 整数 | `÷ 100` | 相对昨收的涨跌百分比。`f170=235` → `2.35%` |
| `f168` | **换手率** (turnover rate) | 整数 | `÷ 100` | 成交量占流通股本比率。`f168=150` → `1.50%` |

#### 标识字段

| 字段 | 中文名 | 说明 |
|------|--------|------|
| `f57` | **股票代码** | 6 位代码，如 `600519` |
| `f58` | **股票名称** | 中文简称，如 `贵州茅台` |

#### 其他字段 (本项目请求但未直接使用)

| 字段 | 中文名 | 说明 |
|------|--------|------|
| `f50` | **量比** (volume ratio) | 当日成交量与过去 5 日均量比值 × 100 |
| `f51` | **涨停价** (upper limit) | 涨停板价格 (整数, 需 ÷ 10^f59) |
| `f52` | **跌停价** (lower limit) | 跌停板价格 (整数, 需 ÷ 10^f59) |
| `f55` | **收益 (每股)** | 每股收益 (EPS)，需额外精度处理 |
| `f116` | **总市值** (total market cap) | 单位: 元 |
| `f117` | **流通市值** (circulating market cap) | 单位: 元 |

### 1.3 示例

请求:
```
GET https://push2.eastmoney.com/api/qt/stock/get
    ?secid=1.600519
    &fields=f43,f44,f45,f46,f47,f48,f57,f58,f59,f60,f168,f170
    &ut=fa5fd1943c7b386f172d6893dbbd1177
```

响应 (简化):
```json
{
  "data": {
    "f43": 152300,
    "f44": 153800,
    "f45": 151200,
    "f46": 152000,
    "f47": 25631,
    "f48": 3904568320.0,
    "f57": "600519",
    "f58": "贵州茅台",
    "f59": 2,
    "f60": 151500,
    "f168": 203,
    "f170": 53
  }
}
```

解读:
- `f59=2` → 精度 2 位小数，除数 = 100
- 最新价: `152300 / 100 = 1523.00` 元
- 最高价: `153800 / 100 = 1538.00` 元
- 最低价: `151200 / 100 = 1512.00` 元
- 开盘价: `152000 / 100 = 1520.00` 元
- 昨收价: `151500 / 100 = 1515.00` 元
- 成交量: `25631 手 × 100 = 2,563,100 股`
- 成交额: `3,904,568,320.00 元 ≈ 39.05 亿元`
- 涨跌幅: `53 / 100 = 0.53%`
- 换手率: `203 / 100 = 2.03%`

## 2. 日 K 线接口

**URL:** `https://push2his.eastmoney.com/api/qt/stock/kline/get`

### 2.1 请求参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `secid` | `1.600519` | 同实时接口，市场代码.股票代码 |
| `fields1` | `f1,f2,f3,f4,f5,f6` | 基础字段组 (元数据) |
| `fields2` | `f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61` | K 线数据字段组 |
| `klt` | `101` | K 线类型: `101`=日K, `102`=周K, `103`=月K |
| `fqt` | `1` | 复权类型: `0`=不复权, `1`=前复权, `2`=后复权 |
| `beg` | `20240101` | 起始日期 (YYYYMMDD)。`0` 表示不限 |
| `end` | `20500101` | 结束日期 (YYYYMMDD) |
| `lmt` | `120` | 返回记录数上限 (从最近日期往前取) |
| `ut` | `fa5fd1943c7b386f172d6893dbbd1177` | 请求令牌 |

### 2.2 返回结构

响应中 `data.klines` 是字符串数组，每条 K 线为逗号分隔的 11 个字段:

```json
{
  "data": {
    "code": "600519",
    "name": "贵州茅台",
    "klines": [
      "2024-06-14,1523.00,1530.50,1535.00,1518.00,25631,3904568320.00,1.12,0.53,8.00,2.03",
      "2024-06-13,1515.00,1522.50,1528.00,1510.00,31200,4750000000.00,1.19,-0.25,-3.80,2.48"
    ]
  }
}
```

### 2.3 K 线字段位置映射

每条记录按逗号分隔后的 11 个字段:

| 位置 | 字段名 | 中文名 | 单位 | 说明 |
|------|--------|--------|------|------|
| `[0]` | **date** | 日期 | — | 格式 `YYYY-MM-DD` |
| `[1]` | **open** | 开盘价 | 元 | 当日第一笔成交价 (前复权后) |
| `[2]` | **close** | 收盘价 | 元 | 当日最后一笔成交价 (前复权后) |
| `[3]` | **high** | 最高价 | 元 | 当日最高成交价 (前复权后) |
| `[4]` | **low** | 最低价 | 元 | 当日最低成交价 (前复权后) |
| `[5]` | **volume** | 成交量 | 股 | 当日累计成交股数 (**注意: 已是股数，非"手"**) |
| `[6]` | **amount** | 成交额 | 元 | 当日累计成交金额 |
| `[7]` | **amplitude** | 振幅 | % | `(最高价 - 最低价) / 昨收价 × 100`。衡量当日价格波动幅度 |
| `[8]` | **change_pct** | 涨跌幅 | % | `(收盘价 - 昨收价) / 昨收价 × 100`。正值=涨，负值=跌 |
| `[9]` | **change_amount** | 涨跌额 | 元 | `收盘价 - 昨收价`。绝对价格变动 |
| `[10]` | **turnover_rate** | 换手率 | % | `成交量 / 流通股本 × 100`。衡量交易活跃度 |

### 2.4 示例解读

```
"2024-06-14,1523.00,1530.50,1535.00,1518.00,25631,3904568320.00,1.12,0.53,8.00,2.03"
```

| 位置 | 原始值 | 解读 |
|------|--------|------|
| [0] `2024-06-14` | — | 2024 年 6 月 14 日 |
| [1] `1523.00` | 开盘价 | 开盘价 1523.00 元 |
| [2] `1530.50` | 收盘价 | 收盘价 1530.50 元 |
| [3] `1535.00` | 最高价 | 盘中最高 1535.00 元 |
| [4] `1518.00` | 最低价 | 盘中最低 1518.00 元 |
| [5] `25631` | 成交量 | 25,631 股 |
| [6] `3904568320.00` | 成交额 | 约 39.05 亿元 |
| [7] `1.12` | 振幅 | 1.12% — 当日价格波动范围 |
| [8] `0.53` | 涨跌幅 | +0.53% — 相比昨日上涨 |
| [9] `8.00` | 涨跌额 | +8.00 元 |
| [10] `2.03` | 换手率 | 2.03% — 当日流通股中 2.03% 发生了交易 |

### 2.5 关于前复权 (fqt=1)

本项目默认使用**前复权 (forward-adjusted)** 数据:
- 前复权是将历史价格按除权除息比例向前调整，使最新价格保持真实值
- 好处: 历史 K 线连续平滑，适合技术分析和趋势比较
- 如果不复权 (`fqt=0`)，遇到分红、送股等事件时，K 线会出现跳空缺口

### 2.6 K 线类型 (klt 参数)

| klt 值 | 含义 | 说明 |
|--------|------|------|
| `1` | 1 分钟 K | 分时线 |
| `5` | 5 分钟 K | |
| `15` | 15 分钟 K | |
| `30` | 30 分钟 K | |
| `60` | 60 分钟 K | |
| `101` | **日 K** | 本项目使用 |
| `102` | 周 K | |
| `103` | 月 K | |

## 3. 股票搜索接口

**URL:** `https://searchapi.eastmoney.com/api/suggest/get`

### 3.1 请求参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `input` | `茅台` 或 `600519` | 搜索关键词 (股票代码或名称) |
| `type` | `14` | 搜索类型: `14` = 沪深股票 |
| `token` | `D43BF722C8E33BDC906FB84D85E326E8` | 搜索接口令牌 |
| `count` | `5` | 最大返回条数 |

### 3.2 返回结构

```json
{
  "QuotationCodeTable": {
    "Data": [
      {
        "Code": "600519",
        "Name": "贵州茅台",
        "MktNum": "1",
        "SecurityTypeName": "沪A"
      }
    ]
  }
}
```

### 3.3 字段说明

| 字段 | 说明 | 取值 |
|------|------|------|
| `Code` | 股票代码 | 6 位数字, 如 `600519` |
| `Name` | 股票名称 | 中文简称, 如 `贵州茅台` |
| `MktNum` | 市场编号 | `0` = 深交所 (SZ), `1` = 上交所 (SH) |
| `SecurityTypeName` | 证券类型 | `沪A` / `深A` / `创业板` / `科创板` 等 |

> 本项目仅保留 `MktNum` 为 `0` 或 `1` 的结果 (即 A 股)，过滤掉港股、美股等。

### 4.1 实时行情 → `quote_snapshots` 表

| API 来源 | 转换 | 数据库字段 | 类型 | 说明 |
|----------|------|-----------|------|------|
| `f43 / 10^f59` | 价格换算 | `price` | Float | 最新价 |
| `f46 / 10^f59` | 价格换算 | `open` | Float | 开盘价 |
| `f44 / 10^f59` | 价格换算 | `high` | Float | 最高价 |
| `f45 / 10^f59` | 价格换算 | `low` | Float | 最低价 |
| `f43 / 10^f59` | 价格换算 | `close` | Float | 收盘价 (= 最新价) |
| `f47 × 100` | 手→股 | `volume` | BigInteger | 成交量 (股) |
| `f48` | 直接使用 | `amount` | Float | 成交额 (元) |
| `f170 / 100` | 百分比换算 | `change_pct` | Float | 涨跌幅 (%) |
| `datetime.now()` | 抓取时间 | `timestamp` | DateTime | 快照时间 |

| `[1]` | float | `open` | Float | 开盘价 (元, 前复权) |

## 5. 补充说明

### 5.1 数据刷新频率

- **实时行情**: 东方财富 push 接口数据延迟约 3~5 秒 (非 Level-2)
- **日 K 线**: 收盘后数据稳定，盘中取到的是截至当时的日内汇总
- **搜索接口**: 实时可用，与交易时间无关

### 5.2 特殊值处理

| 情况 | API 返回 | 程序处理 |
|------|----------|----------|
| 停牌股 | 字段值为 `"-"` 或缺失 | 返回 `0.0` 或 `None` |
| 非交易时段 | `f43` 等返回上一交易日数据 | 正常存储，通过 `timestamp` 区分 |
| 新股首日 | 无昨收价 | `f60` 可能为发行价 |
| 退市/未上市 | `data` 为空 | 抛出 `EastMoneyError` |

### 5.3 关于 token/ut 参数

- `ut` (`fa5fd1943c7b386f172d6893dbbd1177`): 东方财富行情推送接口的通用令牌，公开可用，不涉及用户认证
- `token` (`D43BF722C8E33BDC906FB84D85E326E8`): 搜索接口的公开令牌
- 这两个值是东方财富前端页面中硬编码的，长期稳定，但如果失效需从东方财富网页中重新提取
